    EcoRide est une plateforme de covoiturage écologique, développée avec :

- HTML / SCSS / Bootstrap
- Vite.js pour le build frontend
- PHP + PDO (backend)
- Base de données MySQL
- Système de crédits, filtres de recherche, notation, et trajets écologiques

## Démarrage rapide

1. `npm install`
2. `npm run dev` (si Vite est configuré)
3. PHP via XAMPP (`back/public`)

## Objectif du projet

Créer une application de covoiturage pour la France métropolitaine, avec des trajets notés, filtrables et optimisés pour l'écologie.
